Clone do YouTube

Este é um clone funcional do YouTube, criado com React e a API do YouTube v3.

 Funcionalidades

- Player de vídeo
- Busca de vídeo pelo ID
- Vídeos relacionados (baseado na primeira tag do vídeo)
- Descrição do vídeo
- Nome do canal
- Foto do canal
- Quantidade de likes
- Modo escuro

 Configuração

Para usar este aplicativo, você precisará de várias chaves da API do YouTube v3. Adicione suas chaves em um arquivo `.env` na raiz do projeto, como mostrado abaixo:


REACT_APP_YOUTUBE_API_KEY_FIRST=sua_chave_aqui
REACT_APP_YOUTUBE_API_KEY_SECOND=sua_chave_aqui
REACT_APP_YOUTUBE_API_KEY_THIRD=sua_chave_aqui
REACT_APP_YOUTUBE_API_KEY_FOURTH=sua_chave_aqui
REACT_APP_YOUTUBE_API_KEY_FIFTEEN=sua_chave_aqui


Substitua `sua_chave_aqui` pelas suas chaves da API do YouTube.

 Uso

Para iniciar o aplicativo, execute os seguintes comandos:


- npm install
- npm start


Isso iniciará o aplicativo em `http://localhost:3000`.
